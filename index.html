¿QUÉ APRENDERÁS?

Fundamentos y Tecnologías emergentes

En este curso, te sumergirás en los fundamentos esenciales del desarrollo de aplicaciones web. Aprenderás a construir sólidas bases en HTML, CSS y JavaScript, las piedras angulares de cualquier aplicación web moderna. Además, explorarás las tecnologías emergentes que están dando forma al futuro del desarrollo web, como las Progressive Web Apps (PWAs) y las últimas tendencias en arquitecturas de aplicaciones.

Diseño Interactivo y Experiencia de Usuario (UX) en Desarrollo Web

Descubre el arte de crear experiencias de usuario cautivadoras y funcionalidades interactivas. En este módulo, explorarás principios de diseño centrados en el usuario, prácticas de diseño responsivo y técnicas avanzadas de JavaScript para lograr interfaces web dinámicas. Aprenderás cómo optimizar la experiencia del usuario para dispositivos móviles y escritorio, elevando tus habilidades de desarrollo a un nivel superior.

Despliegue Eficiente y Mantenimiento de Aplicaciones Web

Dominarás las estrategias y herramientas necesarias para llevar tus aplicaciones web desde el entorno de desarrollo hasta la producción de manera eficiente. Aprenderás sobre servicios de alojamiento, configuración de servidores, integración continua y prácticas de seguridad. Además, profundizarás en el mantenimiento de aplicaciones web a largo plazo, abordando actualizaciones, monitoreo y resolución de problemas para garantizar un rendimiento óptimo y una experiencia sin contratiempos para los usuarios.

CONTENIDOS

PRIMER CURSO

En la asignatura de Programación, los estudiantes aprenderán los fundamentos de la programación y cómo desarrollar algoritmos para resolver problemas. Se utilizarán lenguajes de programación como Java, C++ o Python.

La asignatura de Lenguaje de Marcas se centra en enseñar cómo utilizar y estructurar diferentes tipos de marcas, como HTML y XML. Los estudiantes aprenderán a crear documentos web estructurados y formateados.

En la asignatura de Bases de Datos, los estudiantes explorarán cómo diseñar y gestionar bases de datos. Se abordarán temas como modelado de datos, SQL y administración de sistemas de gestión de bases de datos.

La asignatura de Entornos de Desarrollo se centra en proporcionar a los estudiantes las herramientas y prácticas necesarias para desarrollar software de manera eficiente. Se explorarán entornos de desarrollo integrados (IDE) y técnicas de depuración.

La asignatura de Sistemas Informáticos aborda los principios y conceptos fundamentales de los sistemas informáticos. Se explorarán temas como arquitectura de computadoras, sistemas operativos y redes.

SEGUNDO CURSO

En la asignatura de Desarrollo de Interfaces Web, los estudiantes aprenderán a crear interfaces web interactivas y atractivas utilizando tecnologías como HTML, CSS, JavaScript y frameworks front-end.

La asignatura de Desarrollo Entorno Servidor se enfoca en la creación de aplicaciones del lado del servidor utilizando tecnologías como PHP, Node.js y bases de datos relacionales.

En la asignatura de Desarrollo Entorno Cliente, los estudiantes explorarán la creación de interfaces de usuario del lado del cliente utilizando tecnologías como JavaScript, AJAX y frameworks front-end.

La asignatura de Despliegue de Aplicaciones Web se centra en las estrategias y técnicas para implementar aplicaciones web en entornos de producción. Se abordarán temas como hosting, servidores web y escalabilidad.
